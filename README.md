# whereto
VISA requirement data gathering & visualisation

Generates `visa_data.json` which contains VISA requirements per destination by country using [Wikipedia's VISA requirements data](https://en.wikipedia.org/wiki/Category:Visa_requirements_by_nationality).

Produces HTML world map visualisation with single- and multi-passport selection for individuals and groups. 

Limited data is available for disputed territories or regions. 

Updated 22/03/2026.
